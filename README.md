# UTS_WORKSHOP_UI

Project sederhana untuk kebutuhan UTS Workshop UI.

## Isi Project
- `index.html`: halaman utama.

## Cara Menjalankan
1. Buka folder project.
2. Jalankan menggunakan server lokal (misalnya Laragon) atau buka `index.html` langsung di browser.

## Tujuan
- Menyelesaikan tugas UTS mata kuliah Workshop UI.
