# UTS_WORKSHOP_UI

Landing Page untuk kebutuhan UTS Workshop UI.

## Isi Project
- `index.html`: halaman utama.
- `style.css`: layouting halaman.
- `script.js`: logika website (library gsap).
- `images`: kumpulan gambar gambar yang dipanggil.

## Cara Menjalankan
1. Buka folder project.
2. Jalankan menggunakan server lokal (misalnya Laragon) atau buka `index.html` langsung di browser atau bisa juga menggunakan extension live server di vscode.

## Tujuan
- Menyelesaikan tugas UTS mata kuliah Workshop UI.
